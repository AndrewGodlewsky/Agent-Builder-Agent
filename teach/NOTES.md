# Teaching Notes

## User preferences (stated 2026-07-21)

- **Full-width layout** (2026-07-21): the user dislikes narrow centered columns — content must expand across the page (body `max-width: none`, `4vw` side gutters), not sit compressed in the middle. Keep this in any future lesson/reference styling.

## Design system (keep consistent across all lessons)
- Canonical stylesheet: `assets/course.css`; quiz behavior: `assets/quiz.js`; skeleton: `assets/lesson-template.html`.
- **Single-file rule:** each lesson INLINES the contents of course.css and quiz.js into `<style>`/`<script>` tags. The assets folder is the source of truth — edit there, then re-inline.
- Aesthetic: drafting-room field manual. Cool porcelain paper, carbon ink, drafting-blue accent. Bahnschrift (DIN) headings, Georgia body, Consolas data. Signature: engineering title block header + evidence stamps (VERIFIED / DIRECTIONAL / UNCONFIRMED).
- Quiz answer options within a question should have the same word count (no formatting clues).

## Combined edition
- `field-manual-complete.html` (workspace root) = all 11 lessons in one standalone file with a table of contents; lesson cross-links become in-page anchors. Generated by `assets/build-combined.py` — **re-run it after editing any lesson** (`python teach/assets/build-combined.py`).
